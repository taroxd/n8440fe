% \subsection{174　夏休み明けの朝}
\subsection{暑假结束后的早晨}

% 新学期の朝、起きて無意識に隣を見て、誰も居ない事に周は少しだけ残念だと思った。\\
开学第一天早晨，周醒来后无意识地看了看身边，发现没有人后，心里稍稍有些失落。\\

% 昨日泊まる筈だった真昼は周の言葉にキャパシティオーバーを引き起こしたのか、恥ずかしがって夕食後には帰ってしまった。\\
真昼昨晚本打算在这边住一晚的，结果大概是听了周的话后超载了，害羞的不行，吃完晚饭就跑回自己家里了。\\

% 流石にこのタイミングで手出しをするつもりはなかったのだが、言った言葉が言葉だったので意識してしまったらしい。\\
虽然再怎么说，周也没有现在就出手的打算，但毕竟话已经说出去了，让真昼也在意了起来。\\

% 嫌ではなく心を落ち着かせるために帰る、と言って帰宅した真昼に、最初の慰めと不安の解消という目的は達成しているのでそのまま見送ったのだ。\\
真昼说着「不是讨厌，是为了冷静下心情」起身回家，周想到自己一开始安慰和抚平她不安的目的已经达到，便就这样目送她出去了。\\

% あの華奢な体を抱き締めて寝たらさぞ幸せだったんだろうな、と思いながら起き上がって着替えていると、玄関の方からドアの蝶番が軋む音がした。\\
要是能抱着她那娇嫩的身子睡觉，那肯定很幸福的啊——周心里一边这么想，一边起床换着衣服，这时玄关那传来了门锁打开的声音。\\

% 夏休み明け初めての登校日なので、いつもより早めにやってきたのだろう。\\
真昼应该是考虑到今天是过完暑假，开学的第一天，来的也比平常更早。\\

% 制服に着替えて自室から出れば、丁度エプロンを身に付けている真昼がキッチンの入り口に居た。\\
周换上校服走出房间，正好看见身穿围裙的真昼站在厨房的入口。\\

% 「あ……お、おはようございます」\\
「啊……早。早上好」\\

% やや躊躇いがちに挨拶してきた真昼に、周はひっそりと笑った。\\
真昼略显犹豫地打着招呼，周则对她淡淡一笑。\\

% 明らかにこちらを意識しているのが分かる。\\
真昼明显是在意着这边。\\

% 「おはよ。ちゃんと眠れたか？」
「早。睡得还好吗？」

% 「……一応。周くんのせいで別件で悩みましたけどね」
「……还行吧。虽然周君搞得我为别的事情很是苦恼呢」

% 「嫌だったか？」
「讨厌么？」

% 「……分かってる癖にそういう事聞くのやめてください」\\
「……明明知道还问周君真是坏心眼」\\

% 顔を赤くした真昼にぺし、と側にあったキッチンミトンではたかれたので、周は悪びれる事なく笑って、一旦身支度を整えるために洗面所に向かった。\\
真昼羞红了脸，拿起一边的隔热手套开始拍起了周，周则毫不介意地笑了起来，走去洗脸池整理穿着去了。\\

% \\


% 「……何でこちらを見るのですか」
「……为什么总盯着我看啊」

% 「いや、照れてる真昼が可愛いと思って」
「哎呀，想着害羞的真昼真可爱啊就」

% 「周くんがたらしに進化していてよくないと思います」
「周君居然会说谎了看来是学坏了呢」

% 「真昼限定だから問題ない」\\
「只对真昼所以没问题」\\

% 朝食の時に未だに照れている真昼を眺めていたら、真昼が微妙に拗ねる、というよりは照れ隠しに拗ねた振りをしていた。
上了餐桌，周看着还在害羞着的真昼，让真昼稍稍闹起了别扭——准确来说，是为了掩饰害羞，故意摆出一副闹别扭的样子。

% 昨日の事が響いている真昼はこちらを意識している素振りをちょくちょく見せるので、それが可愛くてつい笑ってしまう。\\
真昼脑子里还想着昨天的事，露出一副在意着这边的样子。时不时望见这样的真昼，周觉得很可爱不小心笑了出来。\\

% 余裕を見せる周に真昼は「ばか」と随分と罵声にしては可愛らしい言葉を周に向け、だし巻き玉子を口に放り込む。\\
看着一脸从容的周，真昼骂了周一句「笨蛋」这样颇为可爱的话，然后吃了一口蛋卷。\\

% 微妙に唇が山を築いているのは、周の態度が原因だろう。\\
那微微撅起的嘴唇，大概是给周的态度惹的。\\

% むくれつつも上品な仕草で朝食をとっている真昼が微笑ましくて穏やかな眼差しで眺めていれば、気付いたらしくじと目で見られた。\\
周以安稳的眼神望着以虽显不悦，但依旧很优雅的动作吃着早饭，令人不禁露出微笑的真昼，然后真昼似乎是注意到了，盯了回来。\\

% 「……何ですか」
「……怎么了」

% 「いーや、幸せだなあと。一緒に美味しい朝ご飯をとる事が出来て」
「哎～呀，就是感叹我很幸福嘛。能一起吃好吃的早饭」

% 「……それは私も幸せですけど、今そういう事を言うのはわざとですよね」
「……虽然我也觉得很幸福，但现在你突然这么说总觉得别有深意」

% 「ご想像にお任せしよう」\\
「那就任您想象了咯」\\

% そう言ったら真昼が机の下でつつくように足を蹴ってきたので、羞恥に顔を染めながらご飯を食べる真昼を眺め、周もこれ以上刺激をすまいと静かにご飯を食べるのであった。
周这么说完，真昼便在桌下伸脚踢了周的脚一下。看着满脸害臊地吃着饭的真昼，周便不再刺激真昼，而是静静地吃起了早饭。
