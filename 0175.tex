% \subsection{175　登校}
\subsection{上学}

% 「忘れ物はないですか？」\\
「没有忘记什么东西吧？」

% 朝食を食べて身支度を改めてした後に家を出ようとすると、真昼が声をかける。\\
周吃完早饭，再次整理了下衣服，正打算出家门，突然被真昼叫住了。\\

% 忘れ物、と言われても一応事前準備はしてあるので、恐らくない。
不过周已经做过了事前准备，忘记什么东西应该不大可能。

% 今日は午前中のみの日程なので教科書は必要ない、上履きと筆記用具と課題、提出書類くらいなものだろう。全て先んじて鞄に詰めて確認までしているので、問題ない筈だ。\\
今天只要去上午半天，所以没必要带课本，要带的大概就室内鞋、文具、暑假作业跟一些要交的表格了。这些周都事先装进包里确认好了，所以应该没有问题。\\

% 「特にないと思うけど」
「我觉得应该没啥」

% 「本当に？」
「真的？」

% 「むしろ何でそんな疑うんだ」
「怎么啦总问这个」

% 「……こちらをお忘れでは？」\\
「……是不是忘了这个？」\\

% 若干あきれたように告げつつ真昼が周に見せたものは、暑苦しいから後で締めようと思っていた学校指定のネクタイだ。
真昼一边略显放弃地说着，一边拿给周看的，是学校发的那条周嫌太热打算暂时不系的领带。

% ああ、と思わず声を漏らせばため息をつかれる。\\
周不自觉地啊啊的应到，然后叹了口气。\\

% 「一応休暇明けの集会があるのですから、身なりはしっかりするべきです」\\
「毕竟要参加开学典礼，应该好好整理好仪表」\\

% 全くもう、と言いながらネクタイを首に巻こうとしている真昼に、なんだか面映ゆい気持ちになりつつ軽く屈んでおく。\\
看着真昼一边抱怨着自己一边给自己系着领带，周莫名觉得有些不好意思，稍稍弯了弯身子。\\

% もちろん夏休み前までは基本的に毎日やっていた事なので自分でも出来るのだが、真昼がしてくれるというのなら止めるつもりはなかった。\\
虽说在暑假前这是周每天都要做的事情，周自己自然能完成，但既然真昼在帮自己做，周也没有不让她做的意思。\\

% 大真面目にネクタイを結んでいる真昼の姿に、小さく笑う。\\
看着真昼一脸严肃地系着领带的样子，周微微笑了起来。\\

% （……後で気付いたら照れるんだろうなあ）\\
（……一会反应过来怕是又要害羞了）\\

% 自分で新婚夫婦のような真似をしているのだから、微笑ましい。\\
真昼自己做出宛如新婚夫妇般的举止，令周不禁露出微笑。\\

% 周としてはしてくれる事がありがたいし気遣いもありがたい、当たり前のようにしてくれるのもありがたいし、何より後で気付いた真昼が照れる姿を見られるのが嬉しいので、良いこと尽くしだ。\\
对周来说，不论是为自己系领带，还是关系自己，抑或是这理所当然的行动，自己都十分欢迎，最重要的是之后能看见真昼害羞的样子自己也很高兴，因此是百利而无一害。\\

% せっせとネクタイを結び整えてくれている真昼を眺めていると、視線の質がいつもと違う事に気付いたのか、不審げな眼差しが返ってくる。\\
周看着认真细致地系着领带的真昼，然后真昼大概是发觉了周眼神里的意思变了，便以怀疑的眼光看了回来。\\

% 「……どうかしましたか？」
「……怎么了吗？」

% 「いや、何でもないよ。可愛いなって思っただけ」
「嗯，没什么。只是想着真昼真是可爱」

% 「最近の周くんは可愛いと言ったら誤魔化せると思ってませんか」
「周君最近是不是开始觉得只要说可爱就能把我糊弄过去了」

% 「思ってないけど、可愛いのは事実だからな」
「没啊，不过真昼可爱是事实啊」

% 「……そういう所がたらしだとあれほど……いえ、私限定ならいいですけどっ」\\
「……都说了多少遍这样是在……算了，只对我这样的话那也没事」\\

% 微妙に強気を窺わせる声で言い切って周のネクタイをきっちり締めた真昼に、これ以上はからかうまいと笑うのは控える事にする。\\
看着真昼略为积极地说完，迅速地系好了自己的领带，周料到再说下去就超出玩笑的范畴了，便控制住了自己的笑容。\\

% 代わりに、真昼の頭を一度撫でて、真昼の手を取る。\\
作为代替，周摸了摸真昼的头，然后拉起了她的手。\\

% 「じゃ、行こうか」
「那就走吧」

% 「……誤魔化された感があります」
「……总觉得被糊弄了」

% 「気のせいだ」\\
「错觉啦」\\

% 素知らぬ振りをして、自分の鞄を背負い真昼の鞄を持ちつつ玄関に向かう。
周诳装不知，背起自己的包，然后提起真昼的包，走向了玄关。

% 真昼は自分で持とうとしていたが、彼女に何でもしてもらう訳にはいかないからこれくらいしないと甘えっぱなしになってしまうだろう。\\
虽然真昼想要自己来提，但什么都让她做就实在说不过去了。自己不也做点什么的话，那就成了完全依靠真昼的废人了。\\

% 譲る気はない周に真昼がほんのり嬉しそうに頬を緩めて、周の二の腕に軽く頭突きをする。\\
周不打算让步，让真昼略显高兴，放松了脸颊，然后拿头轻轻撞了下周的上臂。\\

% 「どうした？」
「怎么啦？」

% 「……何でもないです」
「……什么也没有」

% 「何でもなくないと思うけどなあ」
「感觉不像什么都没有呢」

% 「周くんはさっき何でもなかったので、私も何でもないのです」
「因为周君刚刚说了什么也没有，所以我也这么说」

% 「それを言われると追求できないなあ。はいはい、なんでもないんだな」\\
「这么说我就难问了啊。算了算了，什么都没有，好了吧」\\

% 笑って玄関で靴を履き、真昼と一緒に家を出る。\\
周一边笑着一边在玄关穿好了鞋，然后和真昼一起走出家门。\\

% 「行ってきます」\\
「我出门了」\\

% 誰に向けるでもなく呟くと、真昼はじっとこちらを見た後に小さく追従するように「行ってきます」と口にした。\\
周自言自语地轻轻说到。真昼望了望周这边，然后像是跟着周一般，也小声说了一句「我出门了」。\\

% 真昼の返ってくる場所はここなんだな、と思うと嬉しくて頬が緩んでしまうのだが、真昼からの追求はなかった。
周心想着，真昼回到的家是这边啊，高兴了起来，脸也不自觉地放松了，但真昼并没有开口追问。

% なにせ、真昼の方もほんのりと赤らんだ顔で嬉しそうに笑っていたのだ。周の事を言える訳がない。\\
要论原因，是真昼自己也正面带红潮，一脸开心地笑着。根本没理由说周。\\

% 幸せそうにしている真昼の手を握れば、真昼も同じように握り返した。
周握紧了散发着幸福氛围的真昼的手，真昼也同样握紧了周的手。
