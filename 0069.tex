\subsection{69　闲话}

% 起きたら、目の前に彼女が纏うワンピースの布地が見えた。\\
周醒来一睁眼，映入眼帘的便是真昼身上穿着的连衣裙的布料。\\

% どうやら寝返りを打ったらしく、彼女の体側に顔が向いている。
看来是睡觉的时候翻了个身，结果脸就朝着真昼身体这边了吧。

% お陰で柔軟剤の匂いや真昼本来の甘いミルクのような香りがして、顔がじわりと熱を持ち始めるのが分かった。\\
也正是因此，周的鼻子里回荡着柔顺剂与真昼身上那牛奶般的甜味交织的香气，继而明显地感到脸上也渐渐热了起来。\\

% 流石にお腹付近を見続ける訳にもいかず恐る恐る上を向いてみると、慈しむような眼差しで周を見下ろす真昼が居る。
觉得一直盯着人肚子看实在是不像话，于是周战战兢兢地扭头向上看去，便看见真昼以慈祥的眼神低头看着自己。\\

% 失敗した、と思ったのは、やはり真昼は思ったよりもある、というのを見てしまったからだろう。下からのアングルだと質量が強調されるので、とてもいたたまれない。\\
糟啦——周这么想，是看见真昼比自己预想的还要有料所致吧。加之自下而上的视角更凸显了质量感，令周十分的尴尬。\\

% 「おはようございます」
「醒来啦」

% 「……おはよう」
「……嗯」

% 「ぐっすりでしたね」\\
「睡了个好觉呢」\\

% くす、と小さな笑い声をあげる真昼は、からかうというよりは本当に嬉しそうといった風で、まるで元気な子供が寝付いたのを喜ぶ母のような眼差しでもあった。\\
真昼发出轻轻的笑声，表情比起捉弄周更像是真心地感到高兴，眼神甚至让周联想到望着自己那入睡了的可爱孩子而感到幸福的母亲。

% 慈愛の表情で髪を撫で続けている真昼に、周としてはごろごろとのたうち回りたい気持ちで一杯である。
被一脸慈祥的真昼不停地用手抚摸着头，周尴尬到不禁想找个地洞钻进去。

% 寝顔を見られたとかで恥じらう訳ではないが、こんな表情で甘やかすように触れられていたという事を思うと悶絶しそうだ。嫌ではないしむしろ喜んでる自分に恥ずかしさを覚えた。\\
尽管周并没有由于自己的睡相被看到而感觉害羞，但一想到真昼用那样的表情疼爱地抚摸着自己，周便觉得羞愧难当——特别是因为自己对此非但没有觉得讨厌，甚至还很享受其中。\\

% 「……どのくらい寝ていた」
「……我睡了多久了」

% 「一時間ですかね？」
「大概一个小时吧？」

% 「ごめん」
「抱歉」

% 「いえ、周くんの髪を触るの楽しいですし」
「没事的，摸周君的头发很有意思的」

% 「……そうかよ」\\
「……是嘛」\\

% 未だに周の髪を撫でて手触りを堪能しているらしい真昼に、周は直視していられなくて目を逸らした。　\\
周实在是没法继续看着还在享受着摸自己头发乐趣的真昼，于是撇开了目光。\\

% 「周くんは嫌でしたか？」
「周君不喜欢被摸吗？」

% 「……嫌なら寝てないだろ」
「……要不喜欢那也不可能会睡着吧」

% 「そうですか。言ってくれたらいつでもしますからね」
「确实呢。你想的话随是都可以哦」

% 「……そういう事を軽々しく言うなよ」
「……这种事别随口就说出来啊」

% 「どうして？」
「为什么呢？」

% 「……どうしてもだ」\\
「……总之就是不要啦」\\

% 勘違いしてしまいそうになる、なんて言えずに口を閉ざした周に、真昼は首をかしげながらもなでなでと掌を動かし続けている。\\
感觉真昼完全就没有搞明白，周只好闭上了嘴，只留下虽然不解地歪着头，但手却依旧没有停下的真昼。\\

% 「私は、周くんに触るのも触られるのも好きですよ」
「我呢，如果是周君的话，不论是我摸还是摸都挺喜欢的哦」

% （だからそういう事を軽々しく言うなと……）\\
（所以说真的这种事别随口就说出来啊……）\\

% 真昼に他意はない筈だ。
真昼应该是没有别的意思的。

% 本人としては、家庭環境ゆえにスキンシップに飢えていて頭を撫でてもらうと落ち着く、といった意味なのだろうが、聞きようによっては好意があるから触られる事そのものが幸せに繋がるという風にも聞こえるのだ。\\
其实真昼表达的是因为家庭原因，十分渴求亲密的接触，因而被摸头可以安心下来的意思，但话听起来也可以理解成真昼是因为对周抱有好意，所以被摸头这件事本身可以让她感到幸福的意思。\\

% 「……そういう事言ってるとセクハラするぞ」
「……你要这么说我可要性骚扰了啊」

% 「セクハラ」
「心稍牢？」

% 「体に触ったりするぞ」
「就是摸你身体啦」

% 「どこを？」
「想摸哪里？」

% 「どっ、どこって……そりゃその、……言わせるなよ……」\\
「哪，哪里啊……那当然是……我怎么好说出口啦……」\\

% 男子が異性の体のどこを触りたがるなんて、言わずとも分かるだろう。
论起男性想要摸的异性身体的部位，那简直是不言自明。

% つい顔が赤くなってしまう周に、真昼はぱちりと瞬きを繰り返した後に「ああ」と得心がいった風に頷いた。\\
看着脸上呼地红起来的周，真昼呆呆地眨了几下眼，然后总算是有头绪了一般，点着头「啊啊」地说到。\\

% 「触りたいので？」
「想摸是说？」

% 「……交際している女性となら、合意の上で」
「……对象是正在交往的女性的话。在对方同意的前提下」

% 「結局周くんは何もしないって事になりますね」
「结果周君还不是什么都不会做嘛」

% 「うるせえ」
「要你说」

% 「ふ、ふふっ、周くんってすごく律儀で紳士的ですよね」
「呼，呵呵，周君还真是个守规矩的绅士呢」

% 「やかましい」
「多嘴」

% 「周くんのそういうところ、好きですよ」\\
「我很喜欢周君这样的地方哦」\\

% おかしそうに、楽しそうに笑いながら言った言葉に、周は息をつまらせた。\\
觉得很有趣而乐在其中的真昼笑着说出的这话，却让周一瞬间忘记了呼吸。\\

% 本人に、他意はないのだ。
她本人想必是没有别的意思吧。

% この好きというのは人として好ましいの意で言っているのであって、異性としてのものではないだろう。\\
想必，真昼口中的这个「喜欢」，是对朋友的喜欢，而非对异性的喜欢吧。

% それは分かっていたものの、心臓が痛い。
尽管脑中明白，心跳却激烈的生疼。

% そもそも膝枕なんかされてるから勘違いしそうになるのだ。こんな甘やかされては、多少なりと好意を抱かれていると思っても仕方ないだろう。\\
本来光是膝枕就已经让人很容易误会了。加之还被这样疼爱，就算周想成真昼或多或少对自己抱有好意也不足为奇吧。\\

% 実際にある程度好意は持たれているだろうが、それがどんな種類でどのくらいのものなのか分からない周としては、真昼の言葉に一喜一憂してしまって心臓が大変な事になりそうだった。\\
一定程度上真昼对周应该也是抱有好感的，但对搞不明白这好意到底是何种类、程度多深的周来说，真昼这话实在是令人一悲一喜，对心脏很是不好。\\

% 「……そりゃどーも」\\
「……那真是感谢啊」\\

% 精一杯そう言って、周はごろんと寝返りを打って真昼から背を向けた。
周好不容易才把这句话挤了出来，然后翻了个身，把背朝向了真昼。
